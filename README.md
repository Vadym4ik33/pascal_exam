## 🔢 1. Математические функции

| Команда | Что делает | Пример | Результат |
| :--- | :--- | :--- | :--- |
| `Sqr(x)` | Возводит число в **квадрат** | `Sqr(4)` | `16` |
| `Sqrt(x)` | Вычисляет **квадратный корень** (всегда `real`) | `Sqrt(16)` | `4.0` |
| `Abs(x)` | Возвращает **модуль** (абсолютную величину) числа | `Abs(-5)` | `5` |
| `Sin(x)` | Вычисляет **синус** угла в радианах (всегда `real`) | `Sin(0)` | `0.0` |
| `Cos(x)` | Вычисляет **косинус** угла в радианах (всегда `real`) | `Cos(0)` | `1.0` |
| `Pi` | Встроенная константа числа $\pi$ (тип `real`) | `Pi` | `3.141592...` |

## 🔄 2. Округление и преобразование типов

| Команда | Что делает | Пример | Результат |
| :--- | :--- | :--- | :--- |
| `Round(x)` | Округляет дробное число до **ближайшего целого** | `Round(3.6)` | `4` |
| `Trunc(x)` | Просто **отбрасывает дробную часть** (не округляет) | `Trunc(3.6)` | `3` |
| `IntToStr(x)` | Переводит целое число в **текстовую строку** | `IntToStr(123)` | `'123'` |
| `StrToInt(s)` | Переводит текстовую строку в **целое число** | `StrToInt('45')` | `45` |

## 🎛️ 3. Арифметические операторы

| Оператор | Что делает | Применяется к | Тип результата | Пример |
| :---: | :--- | :--- | :--- | :--- |
| `+` | Сложение | Числа / Строки | Зависит от операндов | `5 + 2 = 7` |
| `-` | Вычитание | Числа | Зависит от операндов | `5 - 2 = 3` |
| `*` | Умножение | Числа | Зависит от операндов | `5 * 2 = 10` |
| `/` | **Обычное деление** | Числа | **Всегда `real`** | `5 / 2 = 2.5` |
| `div` | **Целочисленное деление** (без остатка) | Только `integer` | **Всегда `integer`** | `14 div 4 = 3` |
| `mod` | **Остаток от деления** | Только `integer` | **Всегда `integer`** | `14 mod 4 = 2` |

## 📝 4. Ввод, вывод и строки

* `Read(x)` — считывает значение с клавиатуры и оставляет курсор на той же строке.
* `Readln(x)` — считывает значение и **переводит курсор на новую строку**.
* `Write(x)` — выводит данные на экран, оставляя курсор в конце текста.
* `Writeln(x)` — выводит данные и **переносит курсор на следующую строку**.
* `Length(s)` — возвращает текущую **длину строки** (количество символов в ней).

---

## 🧩 5. Логические операторы (Для условий `if` и `while`)

* `and` — логическое **И** (условие верно, только если верны обе части).
* `or` — логическое **ИЛИ** (условие верно, если верна хотя бы одна часть).
* `not` — логическое **НЕ** (переворачивает значение: делает из `True` -> `False` и наоборот).
* `xor` — исключающее **ИЛИ** (верно, только если одна часть `True`, а вторая `False`).
* `<>` — оператор **«не равно»**.
